## 2. 평가지표

```
Score = 0.5 × (1 - NMAE) + 0.5 × FICR
```
- **NMAE**: `|예측 - 실제| / 설비용량`의 평균. 단, 실제 발전량이 설비용량의 10% 미만인 시간대는 평가에서 제외.
- **FICR**: 오차율 기준 계단식 단가 구조.
  - 오차율 ≤ 6% → 단가 4.0
  - 6% < 오차율 ≤ 8% → 단가 3.0
  - 오차율 > 8% → 단가 0.0
  - `FICR = sum(실제발전량 × 단가) / sum(실제발전량 × 4.0)` — 발전량이 큰 시간대일수록 그 시간의 오차가 점수에 더 크게 반영되는 가중 구조

### 10. 계절별 파워커브 - 풍속 구간 및 표본 수 기준 정밀 분석

풍속을 15구간으로 나누어 계절별 평균 발전량과 표본 수를 함께 확인했습니다.

- **중간 풍속(3~10m/s대, 표본 수백~수천 건)**: 세 그룹 모두 겨울이 봄가을·여름보다 뚜렷하게 높음 (예: 그룹1 4.5~5.6m/s 구간에서 겨울 9,482 vs 여름 6,919). 공기밀도 가설과 일치하며, 표본이 충분하고 세 그룹에서 일관되게 나타나 신뢰할 만하다고 판단했습니다.
- **저풍속(~2m/s 이하)**: 봄가을이 오히려 높게 나오는 경우가 있으나, 발전량 자체가 작아 순위가 쉽게 뒤집히는 구간으로 판단했습니다.
- **고풍속(12m/s 이상)**: 겨울 표본이 한 자릿수(심지어 n=1)까지 떨어져, 겨울철 급락처럼 보이는 패턴이 실제 현상인지 표본 부족에 의한 노이즈인지 현재 데이터로는 판단할 수 없다고 결론을 내었습니다.
- 이 발견은 3번 섹션의 8%문턱 진단(오차가 중고출력 구간에 집중)과 겹치는 구간이라, 공기밀도를 feature로 명시적으로 추가할 근거가 됐습니다.

### 11. SCADA 이상치와 그룹 매핑 검증

info.xlsx로 추정한 매핑(VESTAS 1~6호기=그룹1, 7~12호기=그룹2)을 SCADA로 검증하려 했으나, 최초 상관계수가 그룹1 0.0013, 그룹2 -0.0001로 거의 무상관이었습니다. 그런데 같은 데이터의 시계열 그래프는 거의 겹칠 정도로 비슷해서, 이 모순의 원인을 추적했습니다.

`describe()` 결과 `vestas_wtg01_power_kw10m`의 최댓값이 42,512,960 — 터빈 정격용량(3,600kW)을 훨씬 벗어난 값이었습니다. `-100~4,320`(정격×1.2) 범위를 벗어나는 값을 결측 처리한 결과, **이상치 비율은 터빈당 0.03~0.1%에 불과**했지만 이 소수의 값이 상관계수를 완전히 무너뜨리고 있었습니다.

이상치 제거 후 개별 터빈 상관계수를 재계산하니 wtg01~06은 전부 그룹1과, wtg07~12는 전부 그룹2와 더 높은 상관관계(0.8~0.93대)를 보여, 애초 매핑이 맞다는 게 확인했습니다. 최종 그룹 합계 상관계수는 그룹1 0.9654, 그룹2 0.9688입니다(그룹3은 애초부터 0.9489로 정상).
